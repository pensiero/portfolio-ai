# Private Biography Context — TEMPLATE

Copy this file to `content/private-context.md` and replace the placeholders with
your real biography. The real file is git-ignored and is NEVER committed or
web-served — it is read fresh at request time (so edits go live without a restart)
and grounds the assistant's answers.

The assistant answers ONLY from this document, synthesizes across sections, and
cites the section heading it drew from. Keep everything factual, well-structured,
and shareable — leave out anything you don't want surfaced (see Notes). Headings
are the citation surface, so make them clear and meaningful.

## Situations & decisions (for "tell me about a time…")
The single highest-leverage section for interviews: 4–6 concrete moments framed
as situation → action → result, so the assistant can answer behavioural
questions ("tell me about a time you led without authority / made a hard call /
recovered a failing project / handled a low performer") with real specifics
instead of vague inference. Keep each true and brief; cover a range:
- **A hard decision under uncertainty** — *Situation:* … *Action:* … *Result:* …
- **Leading / influencing without authority** — *Situation:* … *Action:* … *Result:* …
- **A failure or setback and what it taught you** — *Situation:* … *Action / result:* …
- **A people call (hiring, a low performer, conflict)** — *Situation:* … *Action:* … *Result:* …
- **A project you turned around or scaled** — *Situation:* … *Action:* … *Result:* …
